# PipelineOS — CRM & Sales Pipeline Tracker

A fully functional CRM (Customer Relationship Management) tool 
built from scratch using HTML, CSS, and JavaScript — no frameworks, 
no backend, no database. Designed to help sales and GTM teams track 
leads, manage pipeline stages, monitor performance, and never miss a follow-up.

## 🔗 Live Demo
> Open the file in any browser to run it instantly

## 🚀 What This Project Does

- 7-stage sales pipeline — Lead → Qualified → Demo → Proposal → Negotiation → Won → Lost
- Interactive Kanban pipeline board with deal cards
- Complete lead database with search, stage, segment and geography filters
- Real-time dashboard with KPIs — pipeline value, win rate, active leads, revenue closed
- Follow-up reminder system with automatic overdue alerts
- Notes history per lead — full conversation tracking with timestamps
- Multi-currency support — INR (₹L), USD ($K), GBP (£K), EUR (€K)
- Analytics dashboard — conversion funnel, segment breakdown, industry chart
- Editable personal profile — name, role, company, avatar colour, app name
- Data auto-saved in browser localStorage — works completely offline
- One-click CSV export for reporting and backups
- Keyboard shortcuts — N to add lead, P for profile, Escape to close

## 💡 Why I Built This

I built PipelineOS to deeply understand how GTM (Go-To-Market) 
and sales teams operate in B2B companies. Instead of just reading 
about CRM tools like Salesforce or HubSpot, I built my own — 
learning pipeline management, sales metrics, lead tracking, 
and product thinking completely hands-on.

## 📋 How to Use

1. Download the `pipelineos-crm.html` file
2. Double-click it — opens instantly in any browser
3. Click your profile card (bottom left) to set your name and company
4. Click **+ Add Lead** to start adding companies to your pipeline
5. Move deals across stages as they progress
6. Set follow-up dates so nothing slips through
7. Check Analytics for your win rate and pipeline health
8. Export to CSV anytime for reporting
